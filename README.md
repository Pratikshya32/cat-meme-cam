# 🐱 MemeCat Cam

A real-time, browser-based AI camera that detects your hand gestures and face expressions using your webcam, matching them instantly to hilarious cat memes!

## Interactive Gestures & Expressions

| Input Trigger | Resulting Cat Meme |
|---|---|
| 👍 Thumbs Up | Crying thumbs-up cat |
| ✊ Fist | Fist bump cat GIF |
| 👌 OK Sign | OK approval cat |
| 🙏 Namaste | Peaceful Namaste cat |
| 🤫 Shh (Index finger up) | Quiet shut up cat |
| 😊 Smile | Happy smiling kitten |
| 😛 Tongue Out | Silly tongue cat |

## Getting Started

1. Visit the live demo link below.
2. Grant webcam permissions when prompted.
3. Wait a few seconds for the browser-based vision models to initialize.
4. Strike your favorite gesture or expression to unlock the corresponding cat meme!

👉 **[Live Demo Site](https://pratikshya32.github.io/cat-meme-cam/)**

## Technology Stack

- **MediaPipe Tasks Vision**: Seamless browser-side AI processing (no server overhead).
- **HTML5 & Vanilla CSS**: Clean, responsive layout.
- **JavaScript (ES6)**: Real-time hand and face landmark classification.
